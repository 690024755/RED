RED 0.9.1

Here are some noteworthy features and fixes available in RED 0.9.1

Execution view handles dynamic changes of execution tree

When execution tree dynamically changes late in the execution the execution view will handle those changes and display nodes accordingly. See Execution view paragraph for more informations.

Content assist displays *.yaml files for variables import

Robot Framework supports YAML variable files. File propositions with *.yaml extension are included in content assistant for variable imports.
